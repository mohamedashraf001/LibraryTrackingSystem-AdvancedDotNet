### **Library Tracking System**

**Advanced .NET 8 + SQL Server + Hangfire + EF Core**

Built a complete library management system using **ASP.NET Core 8**, **EF Core 8**, and **SQL Server**, following clean architecture and best engineering practices.

* Implemented full CRUD features with **Soft Delete** and **Optimistic Concurrency** using `RowVersion`.
* Applied **DDD concepts** including **Value Objects** (Price, Currency) and **Domain Events** (BookCreatedEvent) to enhance domain integrity.
* Integrated **Hangfire** to manage background tasks, including a **recurring job** that cleans soft-deleted books every 3 minutes.
* Added **caching**, **advanced filtering**, and **pagination** to improve performance and user experience.
* Configured **Swagger** for API documentation and interactive testing.
* Organized the project using **layered architecture**, **dependency injection**, and clean code principles.
